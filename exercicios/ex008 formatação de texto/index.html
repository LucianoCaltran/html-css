Principais Formatações

Negrito

Nesta frase temos um termo em negrito usando a tag b (não semântica)

Nesta frase tenho um termo em destaque usando a tag strong

Itálico

Nesta frase temos um termo em itálico usando a tag I (não semântica)

Nesta frase temos um termo em ênfase usando a tag EM (semântica)

Texto Marcado

Podemos criar um texto marcado usando a tag MARK

E no outro parágrafo temos outro texto marcado

Texto grande e pequeno

Temos um texto grande e um texto pequeno nesse parágrafo, utilizando as tags BIG e SMALL

Texto deletado

Podemos marcar um texto como excluído para indicar que o mesmo deve ser lido mas não considerado, utilizando a tag del

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que o mesmo foi adicionado posteriormente, utilizando a tag ins

Existe também o sublinhado com a tag U (não semântica)

Texto Sobrescrito

Para inserir coisas do tipo: x20 + 3 , utilizando a tag SUP

Texto Subscrito

Para inserir coisas do tipo H2O , utilizando a tag SUP
